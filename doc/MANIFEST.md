# 🌌 Project Manifest — Приложение «Гороскоп»


## 🎯 Концепция
**«Гороскоп»** — современное мини-приложение (MAX Mini Apps), которое предоставляет персонализированные ежедневные рекомендации на основе астрологических расчётов.

Приложение помогает пользователям лучше понимать своё состояние, планировать день, улучшать отношения и заботиться о себе. Всё строится на **математических алгоритмах** расчёта положения Луны, фаз, стихий и персонального знака зодиака. Никакой «магии» — только данные и алгоритмы.

## 🧭 Основные экраны и функции

**Шапка**  
Выбор знака зодиака + данные из профиля пользователя (имя, аватар) | Текущая дата | Кнопка меню

**Панель навигации:**  
**Ритм • Любовь • Красота**

### Экран «Ритм» (Энергия и продуктивность)
- Виджет **«Час силы»** — пик личной энергии по расчётам.
- Виджет **«Синергия»** — с какими знаками сегодня лучше общаться.
- **Энергетический статус** — уровень силы в % (на основе Луны).
- Влияние лунной фазы + персональные советы по продуктивности.
- Стихийный вектор дня.

### Экран «Любовь»
- Совместимость с партнёром на сегодня.
- График совместимости на неделю.
- Персональные рекомендации по общению.

### Экран «Красота»
- Персональный бьюти-статус (лучшее время для стрижки, ухода, детокса).
- Рекомендации по типу ухода в зависимости от фазы Луны.
- Дни максимального резонанса.

**Меню приложения:**
- Политика конфиденциальности и пользовательское соглашение
- Информация о владельце ИП (Антонов ...)
- База знаний (как работают алгоритмы)
- Выбор языка
- Поделиться приложением с друзьями

## ⚙️ Техническая основа
Приложение использует 6 независимых расчётных движков:
- `HoroscopeEngine.ts`
- `LoveEngine.ts`
- `BeautyEngine.ts`
- `LunarEngine.ts`
- `PowerHourEngine.ts`
- `SynergyEngine.ts`

Все рекомендации формируются **исключительно математически**, на основе астрономических данных и проверенных астрологических правил.

## 💰 Монетизация
- До 3000 активных пользователей — полностью бесплатно.
- Дальше: ненавязчивая реклама при запуске (для пользователей с низким доходом — минимальная).
- Премиум-подписка (отключение рекламы + расширенные функции) — ≈ 300 ₽/мес.

Ожидаемый доход: от рекламы 0,10–4,75 ₽ за показ + подписки.

## 🌟 Польза приложения для пользователей

Приложение «Гороскоп» выполняет важную социальную и психологическую функцию:

- **Снижает уровень стресса и тревожности** — ежедневные структурированные рекомендации дают ощущение контроля и понимания происходящего.
- **Повышает осознанность и мотивацию** — пользователи лучше планируют день, учитывая свои энергетические пики.
- **Поддерживает саморазвитие и self-care** — помогает вовремя заботиться о себе (уход за телом, эмоциональное состояние, отношения).
- **Улучшает межличностные отношения** — рекомендации по совместимости помогают лучше понимать близких людей.
- **Доступная психологическая поддержка** — для многих молодых людей это удобный ежедневный ритуал, заменяющий мини-коучинг и повышающий уверенность в себе.

Особенно востребовано среди поколения Z и молодых взрослых, которые активно ищут инструменты для баланса в быстром мире.

Развитие российского IT-продукта в сфере wellness и цифрового self-care. Создание удобного и бесплатного инструмента для широкого круга пользователей.

## 🛠 Технологический стек
- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4 (Modern Engine)
- **Icons:** Lucide React
- **Fonts:** Manrope (200-800 weights)
- **Deployment Target:** Mobile-first Web App (PWA)

# Design Manifesto: Cyber-Mystic Minimalism

## 1. Core Aesthetics
- **Base:** Deep Space Black (`#050510`).
- **Atmosphere:** Fixed background orbs (Fuchsia/Indigo) with `blur-[120px]`.
- **Surfaces:** Glassmorphism (`bg-white/3%`, `backdrop-blur-3xl`) with 1px borders (`white/10%`).
- **Geometry:** Ultra-rounded corners (`28px` / `rounded-3xl`).

## 2. Typography System
- **Data:** `font-black`, `tracking-tighter`, `leading-none` (Aggressive & Bold).
- **Labels:** `text-[9px]`, `font-bold`, `uppercase`, `tracking-[0.2em-0.4em]` (Technical & Premium).
- **Contrast:** High contrast between massive numbers and microscopic metadata.

## 3. Color Tokens
- **Power/Time:** Amber-500 (`#f59e0b`) + Glow.
- **Synergy/Emotion:** Pink-500 (`#ec4899`) + Glow.
- **Action/Energy:** Gradient `from-fuchsia-600 to-indigo-700`.

## 4. Interaction & Details
- **Feedback:** `active:scale-[0.98]` for cards, `active:scale-[0.96]` for primary buttons.
- **Accents:** Inner icons in soft-tinted glass containers (`bg-color/10%`).
- **Glow:** Neon box-shadows for progress bars matching their accent color.
