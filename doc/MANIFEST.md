# Project Manifest — Гороскоп

## Стек
React 18 + TypeScript, Vite, Tailwind CSS v4, Lucide React, Manrope

## Проект
PWA-приложение (MAX Mini Apps) с ежедневными астрологическими рекомендациями на основе математических алгоритмов (лунные фазы, стихии, знаки зодиака). Три экрана: Ритм, Любовь, Красота.

## ЦА
Поколение Z и молодые взрослые (18–30 лет), интересующиеся self-care, астрологией и персональными рекомендациями. Россияне, пользователи MAX мессенджер.
