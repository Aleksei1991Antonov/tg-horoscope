# Marketing — MAX • Гороскоп

## Tone & Voice
- Спокойный, поддерживающий, без эзотерики
- Никаких «Вселенная», «судьба», «магия», «ритуалы», «звёзды»
- Язык: русский, без слов «должна», «надо», «обязана»
- Девиз: «Никакой тёмной мистики — только свет, тишина и забота»

## Формат поста в MAX
```
26 мая 2026
Мысль дня: [одно предложение, макс 15 слов]
```
- Фото: VK картинка из Pinterest-подборок
- Кнопка: «Открыть гороскоп»

## Нейминг канала
- Слово «Гороскоп» обязательно
- Микс англ.+русский допустим (quiet. гороскоп, Гороскоп Flow)

- ЦА: девушки 18–30, Россия

## In-App Copywriting

### Принципы
- Максимум 3–5 слов для кнопок и заголовков
- Глаголы в повелительном наклонении, но мягкие («Попробуй», «Открой», «Активируй»)
- Никаких «Вы» — только «ты» (ближе к ЦА 18–30)
- Никаких восклицательных знаков
- Без пассивного залога

### Ошибки и пустые состояния
- Нет партнёра в Love: «Выбери знак партнёра» (кнопка с `+`)
- Нет данных: пишем «Скоро здесь появится твой ритм» — без паники
- Загрузка: не показываем текстовые ошибки, только скелетон/оверлей

### Premium — копирайтинг продажи

Контекст (цена не объявлена — премиум в разработке):

| Элемент | Текст |
|---------|-------|
| Описание премиум | «Полный доступ ко всем ритмам, глубокие расшифровки DNA и приоритетная поддержка» |
| CTA (активация) | «Активировать» |
| Trigger | «Твоя личная карта ресурсов» |

Когда появится цена — писать ценность до цены:
- Плохо: «Купить премиум за 199₽»
- Хорошо: «Открой DNA своих ритмов. 199₽/мес»

## Рекламные креативы (Яндекс Директ + VK Ads)

### Структура объявления

```
Заголовок: 15–50 символов (цепляющий, без вопроса)
Текст: до 90 символов (ценность + CTA)
Кнопка: Установить / Открыть / Узнать
```

### Шаблоны креативов

**Тема 1. Ежедневный ритуал**
```
З. | Просто твой гороскоп
Без магии и судьбы. Только ритмы, энергия и забота — каждый день.
Твой личный ритм. Каждый день.
```

**Тема 2. Самопознание**
```
З. | Узнай свою энергию
Никакой мистики — только астро-алгоритмы, твой знак и актуальный ритм дня.
Твоя энергия сегодня.
```

**Тема 3. Quiet Luxury**
```
З. | Тихая роскошь твоего дня
Спокойный дизайн, никакой рекламы. Твой момент тишины с гороскопом.
Красивый гороскоп.
```

**Тема 4. Для отношений (Love)**
```
З. | Проверь совместимость
Кто твой лучший компаньон сегодня? Графики любви подскажут.
Ваша химия сегодня.
```

### A/B гипотезы для рекламы
1. «Твой гороскоп» vs «Личный ритм дня» — какой заголовок цепляет больше
2. Фото спокойное (Quiet Luxury) vs фото с человеком — кликабельность
3. «Забота» vs «Энергия» — какой триггер конвертирует

### Модель для генерации креативов

GigaChat Pro (Sber) — лучший выбор:

```prompt
Ты — копирайтер для рекламы гороскоп-приложения в Яндекс Директ и VK Ads.
ЦА: девушки 18–30, Россия.
Тон: спокойный, премиальный, без эзотерики (никаких «Вселенная», «судьба», «магия»).
Девиз бренда: «Никакой тёмной мистики — только свет, тишина и забота».

Напиши 3 варианта объявления по шаблону:
- Заголовок (до 50 символов)
- Текст (до 90 символов) — ценность + призыв
- Кнопка (Установить / Открыть)

Тема: [тема креатива]
Уникальное торговое предложение: [что даём пользователю]
```

### Рекламные форматы
- **Яндекс Директ**: текстово-графические (заголовок + описание + картинка)
- **VK Ads**: карусель (3–5 слайдов), клик на «Открыть гороскоп»
- Картинки: из Pinterest-подборок (doc/pinterest-search-queries.md), Quiet Luxury стиль
- Канал как точка входа: пост + кнопка «Открыть гороскоп» → приложение → премиум

## ЦА и гипотезы монетизации (Core TBD)

Данные из GigaChat Pro (синтез по рынку digital в РФ). Core контент в разработке — гипотезы не валидированы.

### Сегменты

| Сегмент | Доход | Готова платить | Доля |
|---------|-------|---------------|------|
| Работающая (22–26) | 45–70 тыс. ₽ | 299 ₽/мес | 60% выручки |
| Студентка с подработкой (18–22) | 20–35 тыс. ₽ | 149–199 ₽/мес | 25% |
| В декрете / молодая мама (24–30) | 15–25 тыс. ₽ | до 199 ₽/мес | 15% |

### Города
- МСК + СПб — 45% покупок, чек выше на 20–30%
- Миллионники — 35%, чек чуть ниже
- Регионы — 20%, чаще выбирают пробный период

### Цифровые привычки (покупали за 6 мес)
- Стриминги — 50%
- Мобильные игры / донаты — 40%
- Онлайн-курсы — 35%
- Приложения для здоровья/медитации — 30%
- Донаты авторам — 25%

### Стоп-факторы
1. Недоверие к астрологии (30%) → митигируем: «алгоритмы, математика, ритмы»
2. Нет привычки платить (25%) → ценность до цены
3. Слишком дорого (20%) → 199₽ как золотая середина
4. Нет уникальной ценности (15%) → контент, которого нет бесплатно
5. Навязчивый UX (10%) → никаких дарк-паттернов

### Триггеры конверсии
- **45%** — утро 7–10 (ритуал)
- **30%** — потребность в поддержке
- **15%** — любопытство к новому
- **10%** — FOMO (ограниченное предложение)

### Решение о покупке
- 70% — импульсивно
- 30% — рационально (сравнение)
- 2–5 дней от первого запуска до покупки (если есть триггер)

### Что должно произойти для оплаты
1. Получила точный прогноз 3+ дня подряд
2. Увидела эксклюзивный контент (DNA-карта) только в премиум
3. Push с ограниченным предложением / скидкой
4. Почувствовала статусность (дизайн, без рекламы)
5. Никаких дарк-паттернов (Apple-style честность)
