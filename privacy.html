Политика конфиденциальности

Настоящая Политика конфиденциальности (далее — «Политика») действует в отношении информации, которая может быть получена от пользователя при использовании мобильного приложения «Любовный радар» (далее — «Приложение»).

1. Какие данные собираются

Приложение не собирает, не хранит и не передаёт персональные данные пользователей третьим лицам. Выбранный пользователем знак зодиака хранится исключительно на устройстве пользователя (localStorage) и может быть удалён путём очистки данных приложения в настройках устройства.

2. Передача данных третьим лицам

Приложение не передаёт какие-либо данные третьим лицам. При нажатии кнопки «Открыть в MAX» пользователь переходит по внешней ссылке в сервис MAX, на который распространяется собственная политика конфиденциальности MAX.

3. Использование cookie и аналитика

Приложение не использует файлы cookie, трекеры и системы аналитики.

4. Безопасность

Вся обработка данных осуществляется локально на устройстве пользователя. Приложение не отправляет и не принимает данные по сети (за исключением перехода по внешним ссылкам по инициативе пользователя).

5. Изменения Политики

Оставляем за собой право вносить изменения в настоящую Политику. Новая версия вступает в силу с момента её размещения в Приложении. Рекомендуем периодически ознакомляться с актуальной версией.

6. Контакты

По вопросам, связанным с обработкой данных, можно обратиться по адресу: rabbithole.help@vk.com
